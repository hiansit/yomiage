「Yomiage」は、Webブラウザ上で動作する高機能なテキスト音声合成（Text-to-Speech）アプリケーションです。OSやブラウザに搭載された標準の**Web Speech API**を利用しており、完全無料で無制限に利用できます。

👉 **[デモページはこちらからお試しください](https://hiansit.github.io/yomiage/)**

## ✨ 主な特徴 (Features)

* **長文の安定再生（チャンク・リレー方式）**
  Web Speech APIによくある「長い文章を渡すと途中で止まってしまう」というブラウザの不具合を回避するため、内部で文章を文（句読点や改行）ごとに自動分割（チャンク化）し、途切れることなく順番にリレー再生する仕組みを備えています。
  
* **再生位置のハイライト追尾**
  テキストエリア内で、現在どの文を読み上げているかが青い選択状態（ハイライト）としてリアルタイムに追尾・表示されるため、「今どこを読んでいるか」を視覚的に見失うことがありません。

* **詳細なコントロール**
  * **音源選択**: ユーザーの環境（OS/ブラウザ）にインストールされているすべての言語や音声をリストから自由に選択可能。
  * **スピード調整**: 0.5倍速のゆっくりした読み上げから、最大4.0倍速の超高速読み上げまで直感的なスライダーで調整可能。
  * **音量調整**: スライダーで0〜100のボリュームコントロールが可能。

* **モダンでプレミアムなUI（Glassmorphism）**
  背景のゆっくりと動くグラデーションアニメーションと、すりガラスのような美しい半透明パネル（Grassmorphism）を採用し、長時間の利用でも飽きがこない洗練されたデザインに仕上げています。

* **サーバーレス（フロントエンド完結）**
  HTML / CSS / JavaScript のみで構築されており、バックエンドのサーバー処理を持ちません。そのため、GitHub Pagesなどで手軽にホストするだけで高速かつセキュアに動作します。

## 🚀 使い方 (How to use)

1. [デモページ](https://hiansit.github.io/yomiage/) にアクセスします。
2. 左上のドロップダウンリストから、読み上げたい音声（例：Google 日本語など）を選択します。
3. 中央のテキストエリア内に、読み上げたいテキスト（長文推奨）を貼り付けます。
4. **「▶ 再生」** ボタンをクリックします。あとは自動的に連続再生され、ハイライトが進行していきます。

## 🛠 技術スタック (Tech Stack)

* HTML5
* CSS3 (Vanilla / Glassmorphism)
* JavaScript (ES6+ / Web Speech API `window.speechSynthesis`)
